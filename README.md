# 🚀 مشروع الشامي ستور - Alshami Store Clone

موقع رقمي متكامل لشحن الألعاب، التطبيقات، والوساطة المالية، مصمم ليعمل كـ تطبيق ويب للموبايل ومبني بالكامل على تقنيات سحابية مجانية وسريعة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **الواجهة الأمامية (Frontend):** HTML5, Tailwind CSS (عبر الـ CDN), Vanilla JavaScript.
* **قاعدة البيانات والباكيند (Backend):** Supabase (PostgreSQL, Auth, Real-time).
* **الاستضافة وإدارة الكود:** GitHub & Vercel.

---

## 📌 الميزات الأساسية المستهدفة
1. **محفظة رقمية ثلاثية العملات:** نظام محاسبي مدمج يحسب رصيد العملات بالـ ($ دولار، TL تركي، ل.س سوري).
2. **تسجيل دخول سريع:** حماية الحسابات وتوفير ميزة تسجيل الدخول بـ Google.
3. **نظام طلبات ذكي:** تتبع لحظي لحالة الشحن (قيد المعالجة، مقبول، مرفوض) مع إشعارات للمستخدم.
4. **أتمتة الشحن (APIs):** دعم الربط مع سيرفرات الموردين للشحن التلقائي، وقسم مخصص للشحن اليدوي.
5. **لوحة تحكم إدارة صارمة:** إدارة المستخدمين، تعديل أسعار الصرف، ومراقبة الـ IPs لحظر الاحتيال.
